Optimal Stopping

About The Experiment

You will participate in a game meant to mimic a basic Optimal Stopping scenario.

In this game you will be presented with a series of sequences, every sequence consists of an undisclosed number of values. For every value a decision needs to be made weather to take the current value or try another one.

Your objective is to end up with the largest sum of chosen values.

Some rules to take into consideration:

  • Values which have been passed can't be obtained again.
  • The ordering of each sequence is random, also the amount of values in each sequence is undisclosed.
  • If you reach the last value you will be forced to settle for it.

Good Luck!

This is an experiment conducted as part of our final project for Human-Machine Interaction course at BIU.
